Onderzoek

Waarom noem je de eerste HTML-pagina; index.html?

Omdat je niet naar een ander bestand verwijst. Meestal is de index.html file de homepagina van je website.

Met welke versie van HTML werken we nu?

Met HTML 5

Met welke versie van CSS werken we nu?

Met CSS 3

Hoe kan het dat een website er net iets anders uit ziet in een andere browser?

Niet alle websites interpreteren de codes in HTML en CSS op dezelfde manier. Hierdoor kunnen kleine veranderingen bestaan in je eindproduct of op een website gebaseert op in welke browser je zit.

Welke broswer is het populairst en hoeveel procent RAM gebruikt deze browser?

Google Chrome is het meest gebruikt. Het percentage RAM dat gebruikt wordt, ligt aan het device waarop de browser gebruikt wordt. Ook het aantal geopende tabs etc. spelen mee.
